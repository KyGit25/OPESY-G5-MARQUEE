This is our groups implementation of creation of an OS Emulator that simulates a marquee!

Members:
Desamito, Hector Francis Seigmund, P.
Maristela, Kyle Gabriel, A.

Recommended system specifications to run:
    1. GCC 9.0+ or MSVC 2019+

Instructions on how to run:
    1. Download the file, "main.cpp"
    2. Open the "main.cpp" file on a compatible IDE that has C++ installed.  
    3. Click the Debug/Run button to compile and execute the file.
    4. Enjoy the program!
